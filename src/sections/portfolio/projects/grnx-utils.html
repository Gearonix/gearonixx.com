@grnx-utils

This is just a set of small utilities and presets for other projects to avoid copying the source code in each project. Includes packages such as eslint configs, CSS presets, react hooks. Custom APIs working with localStorage, apollo-client. The process of building, testing and publishing is automated via github actions and accelerated using nx cloud.
