Adonis Ecommerce

E-commerce platform with authorization, messenger, comments and more, built on Next.js/Nest.js. The user can change the profile, communicate with other users in real time, add products to the cart, filter, search. The application integrates Storybook, Jest and RTL, the architecture is Feature Sliced Design. Be sure to check out Github for a complete understanding of the project!
